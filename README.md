**Simple Chatbot – Short Description**

This Python program is a basic rule-based chatbot that interacts with users through predefined responses. It accepts user input, matches it with specific commands, and replies accordingly until the user types **"bye"** to exit.

**Tools/Concepts Used:**

* Functions (`chatbot()`)
* `while` loop for continuous conversation
* `if-elif-else` statements for response selection
* `input()` for user interaction
* String methods (`lower()`, `strip()`) for input processing

**Features:**

* Greets users
* Responds to common questions
* Handles unknown inputs
* Case-insensitive input matching
* Exit option using `"bye"` command
* Continuous conversation until termination
